作业提交方式及要求

  • 编程作业须独立完成，不允许出现抄袭现象(一经发现做无效处理)，作业通过SmartChair系统提交，具体参见《作业递交系统说明》；编程作业须在规定时间内提交，逾期无法提交，特殊情况请与助教老师说明，另做处理.
  • 编程环境请尽量使用Visual Studio 2013 + Qt 5.5或更高版本，程序编写、调试、测试完毕后，请点击Bulid\Clean按钮，清理不必要的文件（例如：.sdf文件）；请检查是否符合编程规范(特别是代码排版、注释书写、变量命名等)；请修改代码以消除编译窗口内所有的warning和error.
  • 编程完毕后，请将工程文件、源代码等放置在一个目录中，使用压缩软件打包成一个*.zip或*.rar文件，文件命名方式为：学号_姓名_H作业编号(例如：PB16001001_张三_H01.zip或PB16001001__张三_H01.rar)，然后通过SmartChair系统提交.

常用软件
  • Qt: 图形界面开发工具
  • FreeGLUT: 简单的图形界面开发库. 使用和安装: 以Windows 7 x64为例，使用时把glut32.dll放在c:\windows\system32\. 用MS Visual Studio(例如2013)编译时, 把头文件glut.h放在C:\Program Files (x86)\Windows Kits\8.1\Include\um\gl目录, 库文件glut32.lib放在C:\Program Files (x86)\Windows Kits\8.1\Lib\winv6.3\um\x64目录.
  • GLEW: OpenGL Extension Wrangler Library, 开源的跨平台 OpenGL 扩展加载库.
  • GLview: 查看OpenGL实现版本和所支持的扩展的小工具.
  • POV-Ray官方网站: 开源的光线跟踪软件.
  • MeshLab官方网站: 网格模型显示、编辑软件.
